# Расчет ресурсов для системы

## Сеть

1. Пропускная способность - показывает, какой максимальный объем может быть передан в единицу времи (чаще всего секунда)

2. Пинг (RTT - round-trip time) - время, за которое отправленный в сеть запрос достигает адресата и возвращается обратно. Считается хороший пинг - до 100 мс. 

## RAM

1. Тактовая частота - скорость передачи данных между процессором и памятью. По частоте можно посчитать пропускную способность конкретной планки в мегабайтах. 

**Частота * 8Mb**

## CPU

1. Тактовая частота - сколько циклов обработки происходит в секунду. IPC (Instructions Per Cycle) - определяет среднее количество инструкций, выполняемых процессором за один такт

**Количество операций = Тактовая частота процессора * IPC**

2. Количество ядер - это независимый процессор, способный выполнять вычисления, что позволяет ему выполнять несколько задач одновременно.

## Диски

1. Размер

2. Время произвольного доступа - время, за которое диск гарантированно выполнит операцию чтения или записи на любом участке диска. 

3. Количество операций ввода-вывода (IOPS) - показывает какое количество блоков успевает записаться/считаться в секунду.
